小米手机

一、允许自启动
1. 打开手机管家，点击进入“应用管理”
2. 点击“权限”进入授权管理
3. 点击“自启动管理”进入应用列表
4. 找到“跳过广告”并将状态设置为开启

二、禁用电池优化
1. 打开手机管家，点击进入“省电与电池”
2. 点击右上角的“设置”图标进入设置
3. 点击“应用智能省电”进入应用列表
4. 找到“跳过广告”并将后台配置设置为“无限制”

三、锁定应用
1. 点击菜单键/底部上滑悬停，进入多任务界面
2. 长按"跳过广告"卡片，点击锁定图标

华为手机

一、允许自启动
1. 打开“手机管家”
2. 进入“应用启动管理”
3. 找到"跳过广告"，并将开关设置为关闭状态
4. 在弹出对话框中“允许自启动”，“允许关联启动”，“允许后台活动”

二、禁用电池优化
1. 打开手机的系统设置
2. 搜索并进入“电池优化”
3. 点击“不允许”并选择查看“所有应用”
4. 找到"跳过广告"，并设置为“不允许”

三、锁定应用
1. 点击菜单键/底部上滑悬停，进入多任务界面
2. 将"跳过广告"App下拉，在App卡片上方出现锁定标志

VIVO手机

一、允许自启动
1. 打开系统设置并搜索进入“自启动”
2. 找到"跳过广告"，并设置为“允许自启动”

二、禁用电池优化
1. 打开i管家，点击“实用工具”
2. 点击“电池管理”
3. 点击“后台耗电管理”
4. 找到并点击"跳过广告"，设置为“不限制后台用电”

三、锁定应用
1. 点击菜单键/底部上滑悬停，进入多任务界面
2. 点击"跳过广告"右上角的“=”，选择“锁定”

OPPO/一加手机

一、允许自启动
1. 打开系统设置，点击进入“电池”
2. 点击“应用耗电管理”
3. 找到并点击"跳过广告"，将“允许xxx”几个选项全部开启

二、锁定应用
1. 点击菜单键/底部上滑悬停，进入多任务界面
2. 点击"跳过广告"右上角的“：”，选择“锁定”

三星手机

一、允许自启动
1. 打开智能管理器
2. 上滑屏幕,点击“自动运行应用程序”
3. 找到"跳过广告"，并开启选项

二、禁用电池优化
1. 打开系统设置，搜索“电池”进入“优化电池使用量”
2. 点击“没有优化应用程序”并选择“全部”
3. 找到"跳过广告"并将选项关闭

三、锁定应用
1. 点击菜单键/底部上滑悬停，进入多任务界面
2. 长按“跳过广告”App图标
3. 点击“锁定此应用程序”

索尼手机

一、禁用电池优化
1. 打开系统设置，搜索“省电”并进入“省电功能”
2. 点击“未优化”并选择查看“所有应用”
3. 找到并点击"跳过广告"，并设置为“不优化”

谷歌手机

一、禁用电池优化
1. 打开手机的系统设置
2. 搜索并进入“电池优化”
3. 点击“未优化”并选择查看“所有应用”
4. 找到"跳过广告"，并设置为"不优化”
